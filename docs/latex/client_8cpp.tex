 Description\+:~\newline
 this node implements an Action client using the Action\+Lib library from R\+OS. It is used to manage the interaction between the server and the user and to send the goal to the robot. The first part implements an interface to interact with the user and decide which action to perform. The possible actions are\+: set the target coordinates, cancel the actual target, or exit the program. The script is also responsible to retrieve the information about the actual position and velocity of the robot from the \textquotesingle{}/odom\textquotesingle{} topic. Then, it publishes them on the topic \textquotesingle{}/actual\+\_\+info\textquotesingle{}, making them accessible for other nodes within the system. Finally, it implements a service to provide the last coordinates set for the target.
